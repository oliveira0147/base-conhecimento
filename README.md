# Base de Conhecimento

Sistema de gerenciamento de conhecimento desenvolvido com Python e Flask, permitindo o registro e consulta de tutoriais, soluções de problemas e documentações.

## 🛠 Tecnologias Utilizadas

- Python 3.x
- Flask
- SQLAlchemy
- SQLite
- Bootstrap 5
